# Apple II Series Validation and Regression Tests

The purpose of this repo is to collect a number of validation and regression tests for people writing emulators for Apple II series computers, as well as 6502/65c02/65816 CPU chips.

I did not write most of these. Where I can, I will link back to the original

Please tell me of tests you think should be added to this collection!

The criteria for inclusion in this repo is:

* the test has been run on real hardware, and there are test results from this real hardware. This is the baseline against which we aim to make our emulators match.

Each test is in a subdirectory, along with 140K or 800K disk images (or both) that you can boot to run the test.

Each test directory will also show the expected test results, along with a narrative description.

And finally, I'd like to start collecting information on test results across a range of different emulators. But the most important starting point is real hardware.
